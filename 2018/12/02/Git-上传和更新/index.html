Git 上传和更新

Posted on 2018-12-02 | In git

注册了github到现在还没好好用过，这次就来捣鼓一下怎么把代码传上去，也就几步

上传本地代码

先在github上新建一个repository



完成后复制要clone 的链接

选择一个目录右键点击git bash here

把git上初始化的文件clone下来

完成后

cd 文件夹名

接下来依次输入以下命令就可以成功上传了

git add .        注:别忘记后面的. ,此操作是把项目文件夹下面的文件都添加进来)
git commit  -m  "提交信息"  (注:"提交信息"里面换成你需要,如"first commit")
git push -u origin master   (注:此操作目的是把本地仓库push到github上面,此步骤需要你输入帐号和密码)

更新

先查看当前git的状态

git status

更新全部

git add *

提交信息

git commit -m "提交说明"

先拉取当前分支最新代码

git pull

上传 

git push origin master
